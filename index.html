Research Interest

 I work in the field of photogrammetry, geometric computer vision, remote sensing image processing and deep learning theory. Currently, I focus on the following research topics:
  • Robust estimation theory and its applications in remote sensing and robot vision
  • Visual SLAM, Lidar SLAM, feature matching, point cloud registration, loop closure
  • Object detection/tracking from 2D/3D remote sensing data
  • Low-level image processing (dehazing, fusion, color consistency, etc.)
